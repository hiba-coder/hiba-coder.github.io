October, 2022

Graduation Defense

I successfully defended my graduation project and was officially awarded the title of Software Engineer from the National School of Computer Science and Systems Analysis (ENSIAS). The final-year project subject, titled «Performance Benchmarking of Executable Domain-Specific Languages», aimed to support Industry 4.0 by accelerating the reconfiguration process of Reconfigurable Manufacturing Systems (RMS), reducing complexity, effort, and time for industry operators. This was achieved through a multidisciplinary, Model-Based Software Engineering (MBSE) approach. During this project, I was introduced to Model-Driven Engineering (MDE) technologies. My internship concluded with valuable results, which served as the foundation for successfully publishing a research paper at ModelsWard conference.

August 31, 2021

Internship-Maroc Telecom

The goal of this internship, at Maroc Telecom, was to design and develop a web application serving as a communication bridge between the project manager and team members. Team members could input their daily completed tasks for a given project, while the project manager could monitor and track the project's progress in real time. The application was built using the Spring Framework for the back end, Angular for the front end, and various other technologies. This project stemmed from an actual need within the company, which had been relying on traditional or tools like Excel. My internship of two months allows me to contribute to advancing the company’s workflow by delivering a secured and fully developed application.

Academic Project

This document is the result of our work as part of the end of the 2nd year project. The aim of this project was to set up a web application through which the supplier will be able to manage his staff, his products, make his inventory, have the traceability of any transaction and allow his customers to have an idea about the products, place orders without having to go there. This web application makes it possible to create a real interaction between the supplier, his customers and his staff.
